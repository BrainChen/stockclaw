# 股票研究通用框架（自上而下 + 自下而上）

## 1) 研究主线

- **自上而下**：宏观 → 行业 → 公司，适合把握景气轮动
- **自下而上**：商业模式 → 财务质量 → 估值，适合精选个股

## 2) 公司层面五问

1. 公司靠什么赚钱？（收入驱动）
2. 护城河是什么？（品牌、渠道、技术、成本）
3. 盈利是否可持续？（毛利率、费用率、现金流）
4. 预期差在哪？（市场一致预期 vs 真实经营）
5. 估值是否有安全边际？（历史分位、同业比较）

## 3) 行业分析关注点

- 市场空间（TAM/SAM/SOM）与渗透率变化
- 竞争格局（CR3、议价能力、替代风险）
- 产业链利润分配（上游/中游/下游）
- 政策监管与技术迭代节奏

## 4) 输出结论时的规范

- **事实**：来自财报、公告、行业数据
- **判断**：基于假设的概率结论，必须说明前提
- **风险**：列出触发条件，而非笼统“风险存在”

## 5) 一句话总结

研究报告的价值不是复述已知事实，而是识别“预期差 + 催化剂 + 风险回撤比”。
