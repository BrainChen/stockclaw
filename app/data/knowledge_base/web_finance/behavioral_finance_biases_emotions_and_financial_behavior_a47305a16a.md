# 行为金融学：偏差、情绪与金融行为
Language: zh-CN
Original Title: Behavioral Finance: Biases, Emotions and Financial Behavior
Source URL: https://www.investopedia.com/terms/b/behavioralfinance.asp
Search Query: behavioral finance biases

## 核心内容（中文）
行为金融学是一种经济理论，将个人在做出金融选择时的非理性行为归因于心理因素或偏差。[1]

行为金融学是行为经济学的一个子领域，它提出心理影响和偏差会影响投资者和金融从业者的金融行为。此外，这些影响和偏差可以解释各种市场异常，包括股市中的严重上涨或下跌。[1]

行为金融学研究心理影响如何影响市场结果，可用于理解不同行业和部门的结果差异。其中，**心理偏差**的影响是行为金融学研究的关键方面。常见的行为金融要素包括**损失厌恶**、**共识偏差**和**熟悉倾向**。[1]

有效市场理论认为所有股票基于所有可用公开信息定价公平，但行为金融学常驳斥这一观点，因为它未纳入非理性的情绪行为。美国证券交易委员会设有专门针对行为金融学的员工。[1]

行为金融学可从多种视角分析，如股市回报等领域，心理行为常被认为影响市场结果和回报。其分类目的是帮助理解人们为何做出特定金融选择，以及这些选择如何影响市场。行为金融学假设金融参与者并非完全理性且自我控制，而是受心理影响，具有某种正常和自我控制倾向。投资者的心理和身体健康会影响决策，随着健康改善或恶化，其心理状态变化，从而影响对现实问题的理性判断，包括金融问题。[1]

行为金融学通常涵盖五个主要概念：**心理账户**指人们倾向于为特定目的分配资金；**羊群行为**指人们模仿大多数人的金融行为，在股市中常导致剧烈上涨或抛售；**情绪缺口**指基于极端情绪如焦虑、愤怒、恐惧或兴奋做决策，常导致非理性选择；**锚定**指将支出水平 привяз到特定参考，如基于预算或满意度效用；**自我归因**指基于对自己知识或技能的过度自信做选择，常源于特定领域的内在天赋，导致高估自身知识。[1]

## 关键要点
- **行为金融学**研究心理偏差如何导致市场异常，如股市剧烈波动，并挑战有效市场假说。[1]
- 核心概念包括**心理账户**、**羊群行为**、**情绪缺口**、**锚定**和**自我归因**。[1]
- 常见偏差有**确认偏差**（偏好确认已有信念的信息）、**经验偏差**（最近事件影响未来预期，也称最近偏差或可用性偏差）和**损失厌恶**（更重视损失而非收益，导致处置效应）。[1]
- 行为金融学解释投资者非理性，如2008-2009金融危机后许多人退出股市，预期更多经济困难，尽管市场随后反弹。[1]
