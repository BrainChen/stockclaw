# 货币政策传导机制
Language: zh-CN
Original Title: Transmission mechanism
Source URL: https://www.ecb.europa.eu/mopo/intro/transmission/html/index.en.html
Search Query: monetary policy transmission mechanism

## 核心内容（中文）
货币政策传导机制是指货币政策决策影响整体经济尤其是价格水平的过程。该机制的特点是存在**长时滞**、**可变时滞**和**不确定时滞**，因此难以精确预测货币政策行动对经济和价格水平的影响。

下图提供了货币政策决策主要传导渠道的示意图说明。

**官方利率变化**  
中央银行向银行体系提供资金并收取利息。由于其对货币发行的垄断权力，中央银行能够完全决定该利率。

**影响银行和货币市场利率**  
官方利率变化直接影响货币市场利率，并间接影响银行向客户设定的贷款和存款利率。  
对未来官方利率变化的预期会影响中期和长期利率，特别是长期利率部分取决于市场对短期利率未来走势的预期。  
货币政策还能引导经济主体对未来通胀的预期，从而影响价格发展。信誉度高的中央银行能牢固锚定价格稳定预期。在此情况下，经济主体无需因担心通胀上升而提高价格，或因担心通缩而降低价格。

货币政策行动引发的融资条件变化及市场预期调整，可能导致资产价格（如股票价格）和汇率变动。汇率变化可直接影响通胀（如果进口商品直接用于消费），也可通过其他渠道传导。

**影响储蓄和投资决策**  
利率变化会影响家庭和企业的储蓄及投资决策。例如，在其他条件不变的情况下，较高利率会降低为消费或投资融资而贷款的吸引力。  
此外，消费和投资还受资产价格变动的影响，通过财富效应和抵押品价值效应。例如，股票价格上涨时，持有股票的家庭财富增加，可能选择提高消费；反之，股票价格下跌时，家庭可能减少消费。  
资产价格还可通过抵押品价值影响总需求，允许借款人获得更多贷款或降低贷款人/银行要求的风险溢价。

例如，较高利率会增加借款人无法偿还贷款的风险。银行可能减少向家庭和企业提供的资金量，从而降低家庭消费和企业投资。

**导致总需求和价格变化**  
消费和投资变化会改变国内商品和服务需求相对于国内供给的水平。当需求超过供给时，容易产生向上价格压力。此外，总需求变化可能导致劳动力市场和中间产品市场条件收紧或宽松，从而影响这些市场的价格和工资设定。

**影响银行贷款供给**  
政策利率变化会根据银行自身资源（或银行资本）水平不同地影响银行获取外部融资的边际成本。这一渠道在金融危机等逆境中尤为重要，此时资本稀缺，银行更难筹集资本。  
除了关注贷款供给数量的传统银行贷款渠道外，还存在风险承担渠道，即银行提供贷款的相关风险激励受影响。该渠道主要通过两种机制运作：一是低利率提升资产和抵押品价值，加上认为资产价值上升可持续的信念，导致借款人和银行接受更高风险；二是低利率使风险较高资产更具吸引力，因为经济主体寻求更高收益。对于银行而言，这两种效应通常表现为放宽信贷标准，导致贷款供给过度增加。[1][2]
