# 金融投资组合的风险价值（VaR）计算方法

Language: zh-CN
Original Title: How to Calculate Value at Risk (VaR) for Financial Portfolios
Source URL: https://www.investopedia.com/terms/v/var.asp
Search Query: value at risk explained

## 核心内容（中文）

**风险价值（VaR）**是一种量化措施，用于基于概率和市场波动性估计投资组合的潜在损失。它是投资银行和商业银行评估特定时间段内潜在金融损失的重要工具，帮助风险管理者了解投资组合、特定头寸或整个公司的潜在损失概率和程度，从而评估风险敞口并判断资本储备是否充足，指导战略决策并缓解过度风险。[1][2]

VaR建模评估指定时间框架内潜在损失及其发生概率。例如，一家金融机构可能确定某资产有一个月**3% VaR为2%**，表示该资产在一个月内价值下降2%的概率为3%。这一3%概率适用于整个一个月期间，并非意味着3%的单个交易日会发生损失。机构级VaR评估有助于识别所有交易头寸的累积风险，使用VaR数据，金融机构可判断是否拥有足够资本储备覆盖损失，或是否需减少高风险集中持仓。[1][3]

计算VaR的主要方法有三种：**历史模拟法**、**方差-协方差法**（参数法）和**蒙特卡罗模拟法**。历史模拟法考察过去回报历史，按从最差损失到最大收益排序，假设历史回报经验可预测未来结果。该方法简单，通过对过去252个交易日（一年总数）风险因素的百分比变化计算VaR，并应用于当前市场价值生成未来价值场景。[1][2]

方差-协方差法假设收益和损失服从**正态分布**，将潜在损失视为均值的**标准差**事件。该方法适用于分布已知且可靠估计的风险测量，但样本量过小时可靠性降低。蒙特卡罗模拟法则使用计算模型模拟数百或数千次可能的回报迭代，提取损失发生概率（如5%），并揭示其影响。该方法适用于各种风险问题，前提是风险因素的概率分布已知。[1][2]

使用VaR的风险管理优势包括：它是一个单一数字，以百分比或价格单位表示，便于金融专业人士解读和广泛应用；VaR可在不同资产（如股票、债券、衍生品、货币）或投资组合间比较；由于其流行性，许多金融软件工具（如Bloomberg终端）已内置VaR计算。[1]

VaR的局限性在于缺乏标准统计协议，低波动期数据可能低估风险事件发生的可能性和幅度，使用正态分布概率进一步忽略罕见极端事件（如**黑天鹅事件**）。VaR仅表示损失范围的最低风险水平，例如95% VaR为20%意味着每20个周期中约有1个周期损失达20%或更多，但更大损失（如50%）仍符合该评估。2008年金融危机显示，VaR低估了次贷抵押贷款组合的风险，导致机构无法覆盖数十亿美元损失。[1][3]

## 关键要点
- VaR估计特定时间段内给定置信水平下投资组合的潜在损失，主要通过历史模拟法、方差-协方差法和蒙特卡罗模拟法计算。[1][2]
- VaR帮助机构评估累积风险、设定资本储备，并支持跨资产比较，便于风险管理和监管合规。[1][3]
- 历史模拟法基于过去回报排序，简单但依赖历史数据准确性。[1]
- VaR局限包括低估极端事件，且不衡量最坏情况损失，如2008危机所示。[1]

## 风险与边界
- VaR假设历史模式或分布可预测未来，但低波动期数据或正态分布假设可能严重低估黑天鹅事件风险。[1][3]
- VaR仅提供损失阈值概率，不覆盖超出阈值的极端损失幅度，手动计算大型组合计算量巨大。[1]
