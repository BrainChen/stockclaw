# 信用利差 - 概述、计算方法、示例

## 核心内容（中文）
信用利差是指到期日相同但信用评级不同的两种债务工具收益率（回报）的差值。换言之，利差是由于信用质量不同而导致的回报差异。[6]

例如，如果5年期国债收益率交易于3%，而5年期公司债券收益率交易于5%，则信用利差为2%（5% – 3%）。

利差用于反映投资者为承担额外信用风险所需获得的额外收益率。信用利差通常使用相同到期日国债与公司债券的收益率差。作为无风险资产（因由美国政府担保），国债可用于衡量公司债券的风险程度。

例如，如果国债票据或债券与公司债券之间的信用利差为0%，则意味着公司债券提供与国债相同的收益率，且无风险。利差越高，公司债券风险越大。

注意：公司债券与国债的到期日必须相同。此外，投资者常以自选基准债券收益率替代国债收益率，例如使用AAA评级公司债券收益率作为基准。

信用利差并非静态，会随时间收窄或扩大，这种变化通常归因于经济状况。例如，在市场状况恶化时，投资者倾向购买美国国债并卖出公司债券持仓。资金流入国债将推高国债价格并压低收益率，而公司债券资金外流将压低价格并抬高收益率，导致国债与公司债券间的信用利差扩大。

相反，在市场状况改善时，投资者倾向购买公司债券并卖出美国国债，因为改善的市场环境下公司债券信用风险较低。资金流入公司债券将推高价格并压低收益率，而国债资金外流将压低价格并抬高收益率，导致信用利差收窄。

投资者欲判断美国经济状况。历史上，2年期BBB评级公司债券与2年期美国国债的平均信用利差为2%。当前2年期BBB评级公司债券收益率5%，2年期美国国债收益率2%。当前信用利差为3%（5% – 2%）。鉴于历史平均为2%，这可能表明美国经济显示出经济疲软迹象。

## 关键要点
- **信用利差定义**：相同到期日但不同信用评级的债务工具收益率差，反映信用风险补偿。
- **计算基准**：常以国债（无风险）收益率减去公司债券收益率，利差越高风险越大。
- **动态变化**：经济恶化时利差扩大（投资者避险转向国债），改善时利差收窄。
- **应用洞察**：利差变动可指示经济状况，如高于历史平均可能信号经济疲软。

## 风险与边界
- 必须确保比较债券到期日相同，否则利差计算无效。
- 投资者可替换国债为其他基准（如AAA债券），但需一致性以准确反映纯信用风险。
