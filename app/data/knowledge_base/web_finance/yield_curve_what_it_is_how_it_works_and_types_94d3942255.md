# 收益率曲线：定义、工作原理及类型

## 核心内容（中文）
收益率曲线是相同信用质量但不同到期日的债券收益率或利率的图形表示，通常通过绘制给定区间内不同到期日的债券收益率来描绘，例如按月划分。

收益率曲线的斜率可预测利率方向以及可能随之而来的经济扩张或收缩。美国财政部网站每日交易日发布收益率曲线利率。

收益率曲线不断变化，受通胀、投资者情绪和央行政策影响。它是市场其他债务的基准，如抵押贷款利率和银行贷款利率，并可预测经济产出和增长变化。

最常报告的收益率曲线比较美国国债的3个月、2年、5年、10年和30年期限。美国财政部利率网站每日交易结束前东部时间下午6点提供收益率曲线利率。

正常收益率曲线显示短期债券收益率较低，随着到期日延长而上升，曲线向上倾斜。这反映经济扩张期，长期债券收益率持续上升。

倒挂收益率曲线向下倾斜，短期利率高于长期利率，常出现在衰退期，投资者预期长期债券收益率下降。在经济低迷时，投资者偏好长期债券，推动其价格上涨并压低收益率，倒挂曲线历史上是衰退预警信号。

平坦收益率曲线显示所有到期日收益率相似，暗示经济不确定性。中期到期（如6个月至2年）可能略高，形成轻微驼峰。投资者在高不确定性时期要求各期限相似收益率。

## 关键要点
- 收益率曲线绘制债券利率随时间变化，通常长期收益率高于短期，正常曲线预示经济扩张，倒挂曲线暗示衰退。
- 正常曲线向上倾斜，低短期收益率向高长期收益率上升，伴随经济扩张、高通胀和高利率。
- 倒挂曲线罕见，短期利率超长期，常在衰退前出现，投资者转向长期债券压低其收益率。
- 平坦曲线收益率相似，显示经济不确定，中期可能有轻微驼峰。

## 风险与边界
- 收益率曲线风险指利率变动对固定收益工具（如债券）回报的不利影响，债券价格与利率呈反比关系。
- 内容基于美国国债示例，中国市场收益率曲线行为可能不同，受本地经济、货币政策影响。
