# 如何计算投入资本回报率 (ROIC)

Language: zh-CN
Original Title: How to Calculate Return on Invested Capital (ROIC)
Source URL: https://www.investopedia.com/terms/r/returnoninvestmentcapital.asp
Search Query: return on invested capital explanation

## 核心内容（中文）

投入资本回报率（**ROIC**）通过将税后净营运利润除以投入资本来计算，显示公司如何高效利用资本产生利润。[1][2]

将公司**ROIC**与其加权平均资本成本（**WACC**）进行比较，可以揭示公司投入资本的使用是否有效。如果**ROIC**超过**WACC**，则公司创造价值。[1][2][3]

除了**ROIC**，企业还使用其他指标评估资本利用情况，如经济增加值和资本使用回报率。投入资本指公司通过发行证券向股权股东和向债券持有人发行债务所募集的资金。[1]

**ROIC**公式为：**ROIC = NOPAT / 投入资本**，其中**NOPAT**为税后净营运利润。另一种表述为：**ROIC = (净收入 - 股息) / (债务 + 权益)**。[1][2]

计算投入资本有多种方法：一种是从总资产中减去现金和非息流动负债（如税负负债和应付账款）；另一种是将公司权益账面价值与债务账面价值相加，再减去非营运资产（如现金等价物、市场证券和停业操作资产）；还有一种是从流动资产减去流动负债得到营运资本，再减去现金得到非现金营运资本，最后加固定资产。[1]

**NOPAT**计算方法包括从净收入中减去股息，或更优选从营运利润调整税负：**NOPAT = 营运利润 × (1 - 有效税率)**。**ROIC**高于资本成本表明公司健康增长，反之则暗示商业模式不可持续。[1][2]

**ROIC**以百分比表示，通常为年化或过去12个月值，应与公司资本成本比较。若**ROIC**高于**WACC**，公司创造价值并以溢价交易。常见价值创造基准为超过公司资本成本2个百分点。[1][3]

零回报公司虽不破坏价值，但无多余资本投资未来增长。比较同行业公司**ROIC**，持续高于同行表明公司运营更好、更盈利。对成熟公司，可比较当前与历史**ROIC**。[1][5]

**ROIC**为其他指标如市盈率（**P/E**）提供上下文。孤立查看**P/E**可能误判公司被低估，但若**ROIC**下降，则可能不再为股东创造价值。高**ROIC**公司值得溢价交易。[1]

例如，Target Corp.（TGT）在其10-K报告中直接计算**ROIC**：从营运收入加净其他收入得**EBIT**，加回营运租赁利息减所得税得**NOPAT**；投入资本包括股东权益、长期债务和营运租赁负债，减去现金等价物。[1]

## 关键要点
- **ROIC** = **NOPAT / 投入资本**，衡量公司资本利用效率，若高于**WACC**则创造价值[1][2][3]。
- 投入资本计算多种方式，包括总资产减非息负债，或权益+债务减非营运资产[1]。
- **NOPAT** = 营运利润 × (1 - 税率)，优于简单净收入减股息，避免一次性收入干扰[1][2]。
- **ROIC**高于资本成本2%为价值创造基准，持续高于行业平均表明优秀运营[1][5][6]。

## 风险与边界
- **ROIC**不区分常规运营收入与一次性活动，可能高估真实效率[1][4]。
- 计算投入资本和**NOPAT**方法多样，若不一致可能导致比较偏差[1][6]。
