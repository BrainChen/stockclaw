# 尾随市盈率 (Trailing P/E)：定义与示例

Language: zh-CN
Original Title: Trailing Price-to-Earnings (Trailing P/E): Definition and Example
Source URL: https://www.investopedia.com/terms/t/trailingpe.asp
Search Query: what is price to earnings ratio

## 核心内容（中文）
尾随市盈率（Trailing P/E）是一种财务指标，用于显示公司当前股价与其过去一年每股收益（EPS）的关系，为投资者提供回顾性估值衡量。[1][6]

尾随市盈率是一种相对估值倍数，基于过去12个月的实际收益计算，通过将当前股价除以过去12个月的尾随EPS得出。[1][2][6]

与**前瞻市盈率（Forward P/E）**不同，后者使用预计未来收益计算市盈率，而尾随市盈率依赖实际报告数据，因此被视为更可靠，但过去收益并非总是未来收益的良好预测，需要谨慎使用。[1][2][4]

市盈率（P/E ratio）通过将公司股价除以最近财政年度收益计算，当泛指P/E时，通常指尾随P/E，即当前市值或股价除以过去12个月EPS。[1][6]

最近财政年度收益可在年度报告的损益表中找到，损益表底部显示公司全年总EPS，将当前股价除以此数即得尾随P/E比率：**尾随P/E = 当前股价 / 尾随12个月EPS**。[1]

## 关键要点
- 尾随P/E用于标准化并比较不同时期和公司间的相对股价，是寻找市场相对便宜股或判断股价过高/过低的常用指标。[1]
- 分析师青睐P/E因其提供收益的“苹果对苹果”比较，但股价波动而收益固定，因此尾随P/E使用最近四个季度收益以保持更新。[1]
- 示例：股价50美元、尾随12个月EPS为2美元，则尾随P/E为25倍，即股价为收益的25倍；若股价降至40美元，则P/E降至20倍。[1]
- 若最近两季度收益下降，可用最新两季度替换年初数据计算，若收益趋势下行，P/E将高于20倍，表明股价可能高估。[1]

## 风险与边界
- 尾随P/E基于过去实际收益，可能不准确反映当前或未来收益情况，尤其当公司有一次性费用影响过去12个月收益时。[1][2]
- 前瞻P/E使用未来收益预测，更相关但易受分析师偏差或公司低估收益以打共识预期的误算影响。[1]
