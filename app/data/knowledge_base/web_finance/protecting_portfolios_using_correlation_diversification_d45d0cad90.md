# 使用相关性多元化保护投资组合
Language: zh-CN
Original Title: Protecting Portfolios Using Correlation Diversification
Source URL: https://www.investopedia.com/articles/financial-theory/09/uncorrelated-assets-diversification.asp
Search Query: portfolio diversification correlation

## 核心内容（中文）
多元化自然吸引每位投资者内心的风险厌恶本能。将所有资金押注于一匹马似乎比分散押注于四匹不同马匹更冒险，而且确实如此。

但是如何选择这些马匹？可以使用直觉随意挑选四匹，但这如同玩机会游戏。专业基金经理不会仅凭直觉挑选良好多元化的投资组合。他们使用统计技术寻找所谓的“非相关资产”。

非相关资产有助于多元化投资组合并管理风险，这对警惕不确定性的投资者是好消息。但这并非完美：通过挑选非相关资产多元化投资组合并不总是有效。本文将说明相关性是什么，并解释非相关资产如何运作——以及何时失效。

**关键要点**  
投资中的多元化概念指拥有多种证券，跨越几个资产类别，以分散风险。历史上，股票和债券被用作两个非相关资产类别的例子。多元化在资产相互非相关或负相关时效果最佳，这样当投资组合某些部分下跌时，其他部分会上涨。

**数字游戏**  
相关性统计衡量两个变量之间关系的程度，用介于+1.0和-1.0之间的数字表示。在多元化投资组合中，相关性代表组合中不同资产价格变动之间的关系度。相关性+1.0表示价格同步变动；-1.0表示价格反向变动；0表示资产价格变动非相关，即一资产的价格变动对另一资产无影响。

实际中，很难找到完美正相关+1.0、完美负相关-1.0或完美中性相关0的资产对。不同资产对的相关性可能介于+1.0和-1.0之间（如+0.62或-0.30）。每个数字显示你距离完美非相关0有多远。例如，若资产A与B的相关性为0.35，A与C为0.25，则A与B的相关性高于与C。

若两组资产提供相同回报和相同风险，选择相关性较低的那组可降低投资组合整体风险。

**并非所有资产相同**  
某些植物在雪顶山上茁壮成长，有些在荒漠，有些在雨林。正如不同天气对不同植物影响不同，宏观经济因素对不同资产的影响也不同。宏观经济环境变化对不同资产的影响各异。例如，由于通胀，金融资产（如股票和债券）与实物资产（如黄金）价格可能反向变动。高通胀可能推高黄金价格，但导致金融资产价格下跌。

**使用相关性矩阵**  
统计学家使用价格数据分析两资产过去价格相对变动情况。每对资产分配一个数字代表其价格变动的相关程度。此数字可用于构建不同资产的“相关性矩阵”。相关性矩阵以表格形式呈现资产间相关性，便于选择不同资产。

在为投资组合选择资产时，需要从众多排列组合中挑选。无论如何组合，多资产投资组合中有些资产正相关，有些负相关，其余相关性散布在零附近。从宽泛类别入手（如股票、债券、政府证券、房地产等），再细化到子类别（如消费品、制药、能源、技术等），最终选择具体资产。选择非相关资产的目标是分散风险，确保单一意外不会摧毁整个投资组合。

## 关键要点
- 多元化指跨越多个资产类别拥有多种证券，以分散风险，股票和债券历史上为例证。
- 相关性衡量资产价格变动关系，+1.0为完全同步，-1.0为完全反向，0为非相关。
- 非相关或负相关资产使多元化最佳，当部分资产下跌时其他上涨，降低整体风险。
- 使用相关性矩阵表格分析资产间关系，从宽泛类别逐步细化选择非相关资产。

## 风险与边界
- 实际中完美相关性（+1.0、-1.0或0）罕见，相关性可能介于之间，多元化效果不总是完美。
- 即使挑选非相关资产，宏观事件仍可能导致资产突然相关，单一风险无法完全消除。
