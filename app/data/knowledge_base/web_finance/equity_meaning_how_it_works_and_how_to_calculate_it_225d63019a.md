# 权益：含义、工作原理及计算方法
Language: zh-CN
Original Title: Equity: Meaning, How It Works, and How to Calculate It
Source URL: https://www.investopedia.com/terms/e/equity.asp
Search Query: return on equity explanation

## 核心内容（中文）
权益是指资产或投资在扣除或偿还任何欠债后的剩余价值；该术语也用于指代用于融资的资本或品牌的价值[1]。

在金融领域，**权益**指一种所有权，其价值会因相关债务而减少。对于房主，房屋权益指房产价值减去任何抵押贷款或债务余额。在股票市场，**股东权益**（或私有公司所有者权益）代表公司资产与负债之间的差额。如果公司所有资产被清算并用于偿还债务，股东权益即为剩余金额。在收购情况下，它是公司销售价值减去随销售不转移的公司负债[1]。

此外，股东权益可代表公司的账面价值。权益有时可作为实物支付形式。它也代表公司股份的按比例所有权。权益可在公司资产负债表上找到，是分析师评估公司财务健康的最常见数据之一[1]。

通过比较公司拥有的和欠下的具体数字，“资产减负债”的股东权益公式清晰描绘了公司财务状况，便于投资者和分析师解读。权益作为公司筹集的资本，用于购买资产、投资项目和资助运营。公司通常通过发行债务（贷款或债券形式）或权益（出售股票）筹集资本。投资者通常青睐权益投资，因为它提供分享公司利润和增长的机会[1]。

权益重要，因为它代表投资者在公司中的权益价值，由其股份比例体现。持有公司股票赋予股东资本利得和股息潜力。权益所有权还赋予股东对公司行动和董事会选举的投票权。这些权益所有权益处促进股东对公司的持续兴趣[1]。

股东权益可为正或负。若为正，公司资产足以覆盖负债；若为负，公司负债超过资产，长期如此视为资产负债表资不抵债。通常，投资者视股东权益为负的公司为高风险或不安全投资。股东权益单独并非公司财务健康的决定性指标；结合其他工具和指标，投资者可准确分析组织健康[1]。

## 关键要点
- **权益**代表若公司所有资产清算并偿还所有债务，返回给股东的价值[1]。
- 权益计算公式为：**股东权益 = 总资产 - 总负债**，用于多项关键财务比率，如 **ROE**（权益回报率）[1]。
- 股东权益可在资产负债表上找到，反映公司净值和所有权比例[1]。
- 房屋权益是房产价值净债务后的另一种权益形式[1]。

## 风险与边界
- 负股东权益表示资不抵债，视为高风险投资，但需结合其他指标评估[1]。
- 权益计算基于会计等式，仅反映账面价值，不完全体现市场动态或未来增长[1]。
