# 股东权益回报率 (ROE) - 公式、示例及指南

## 核心内容（中文）
**股东权益回报率 (ROE)** 是衡量公司年度回报（净利润）除以其总股东权益价值的指标，以百分比表示（如 12%）。[1][2][3]  
ROE 也可通过公司股息增长率除以其收益留存率（1 - 股息支付比率）来计算。[1]  
ROE 是一个双重比率，因为它将损益表与资产负债表结合，将净利润与股东权益进行比较。该指标代表股权资本的总回报，并显示公司将股权投资转化为利润的能力，即每美元股东权益产生的利润。[1][3]  
ROE 的公式为：**ROE = 净利润 / 股东权益**。该指标提供了一个简单的方法来评估投资回报，通过将公司 ROE 与行业平均水平比较，可发现公司的竞争优势，并洞察管理层如何利用股权融资来发展业务。[1][2]  
持续上升的 ROE 表示公司擅长创造股东价值，因为它能明智地再投资收益，从而提高生产力和利润；相反，下降的 ROE 可能意味着管理层在再投资资本于非生产性资产时做出错误决策。[1]  
虽然简单公式为净利润除以股东权益，但 ROE 可进一步分解为资产回报率 (ROA) 和财务杠杆的影响，二者将在下文详细讨论。以净利润作为分子，ROE 考察公司底线盈利能力，为所有者和投资者提供衡量，在支付强制性义务和业务再投资后剩余的超额利润。[1][3]

## 关键要点
- ROE 帮助投资者评估资金回报是否良好，公司评估股权利用效率，必须与公司历史 ROE 和行业平均值比较，不能孤立看待。[1][2]  
- 高 ROE 可能表示公司内部盈利成功，但需注意债务放大效应，如股权 15 万美元、债务 85 万美元，总资本 100 万美元，债务利息 4.2 万美元，若 ROCE 达 12%，剩余利润 7.8 万美元，可使股权回报放大超 50%。[1]  
- 股份回购会扭曲 ROE，因为减少流通股数量缩小分母，从而提高比率。[1]  
- ROE 计算可能排除无形资产（如商誉、商标），导致与其他包含无形资产的公司难以比较；股东权益可取期初、期末或平均值，净利润可替换为 EBITDA 或 EBIT，并调整非经常性项目。[1]

## 风险与边界
- 高 ROE 不完全反映风险，可能因高负债而放大，但增加财务风险。[1][3]  
- ROE 受会计政策和资本结构影响，需结合其他比率全面评估，避免孤立使用。[1][2]
