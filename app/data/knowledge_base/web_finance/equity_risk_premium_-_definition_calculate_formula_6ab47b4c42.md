# 股权风险溢价 - 定义、计算、公式

## 核心内容（中文）
股权风险溢价（**Equity Risk Premium**）是股票回报与无风险回报率之间的差额。无风险回报率通常以长期政府债券为基准，假设政府违约风险为零。它是股票持有人因承担风险而获得的超额回报，即投资者选择股票而非无风险证券的补偿。[1][3]

股权风险溢价与风险水平正相关。风险越高，股票回报与无风险利率之间的差距越大，从而溢价越高。实证数据验证了这一概念，证明长期来看，投资者承担更高风险将获得相应回报。[1][3]

对于理性投资者，投资相关风险增加必须伴随潜在回报上升，否则投资不可行。例如，若政府债券回报率为6%，理性投资者仅在股票回报超过6%（如14%）时选择股票，此处**14% - 6% = 8%**即为股权风险溢价。[1]

无风险资产是指未来回报确定的资产，源于对发行人的完全信心。政府证券被视为无风险资产，因为政府理论上不会违约，可通过央行借款或印钞自筹资金。因此，无风险资产违约风险为零，通胀风险可忽略。[1]

计算股权风险溢价（市场层面）的公式为：**Equity Risk Premium = 股票市场回报率 - 无风险利率**。市场回报率可取相关交易所指数（如美国道琼斯工业平均指数），无风险利率常取长期政府证券当前利率。[1]

**资本资产定价模型（CAPM）**用于基于系统风险（由Beta系数衡量）计算证券预期回报。CAPM中，回报包括两部分：无风险利率（时间价值）和风险溢价（风险补偿）。公式为：**Ra = Rf + βa (Rm - Rf)**，其中Ra为证券a预期回报，Rf为无风险利率，Rm为市场预期回报，βa为股票a的风险系数（=市场回报与股票回报协方差 / 市场回报方差）。股权风险溢价 = **Ra - Rf = βa (Rm - Rf)**。[1]

例如，10年期政府债券回报7%、证券A的Beta为2、市场回报12%，则股权风险溢价 = **2 × (12% - 7%) = 10%**。[1]

## 关键要点
- **股权风险溢价**是股票回报超过无风险利率的超额部分，作为风险补偿。
- 与风险正相关：风险越高，溢价越大，实证数据支持长期回报。
- 计算公式：**市场回报率 - 无风险利率**，市场回报常用指数，无风险率用长期国债。
- **CAPM**整合股权风险溢价：**Ra = Rf + βa (Rm - Rf)**，Beta衡量系统风险。
- 无风险资产以政府证券为代表，违约风险为零。
- 示例：Beta=2、市场12%、无风险7%，溢价=10%。

## 风险与边界
- 股权风险溢价计算依赖历史数据或市场指数，不同时期或指数选择可能导致结果差异[4]。
- CAPM假设市场有效、Beta稳定，实际中Beta波动及非系统风险可能影响准确性[1]。
