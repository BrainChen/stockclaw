# 个体风险 - 定义、类型、示例

## 核心内容（中文）
**个体风险**（Idiosyncratic risk），有时也称为**非系统风险**（unsystematic risk），是指投资特定资产（如股票）所固有的风险。这种风险特定于某一具体投资，与影响整个市场或整个投资组合的风险相对立。它是**系统风险**（systemic risk）的对立面，后者会影响给定资产类别中的所有投资。系统风险包括利率变化或通胀等因素。[2]

个体风险源于个别公司（或个别投资）。投资者可以通过**多元化**投资组合来缓解个体风险。[2]

与个体风险相比，系统风险涉及影响资产（如股票）及其底层公司的因素，在微观经济层面产生影响。这意味着个体风险与整体市场风险的相关性很小或几乎没有。最有效的缓解或消除个体风险的方法是通过投资多元化。[2]

个体风险本质上是不可预测的。研究显示，个股随时间面临的大部分风险变动是由个体风险造成的。如果投资者希望减少这种风险对其投资组合的潜在剧烈影响，可以通过**多元化**和**对冲**等投资策略实现。多元化策略涉及投资多种低相关性资产，即市场中通常不同步波动的资产。其理论基础是，当一两个资产亏损时，投资者其他非相关投资的收益可对冲或最小化损失。[2]

相反，系统风险涉及宏观经济因素，不仅影响单一投资，还影响整体市场和经济。增加投资组合中的资产或多元化资产无法抵消系统风险。[2]

公司及其股票面临各自固有的风险。个体风险的最常见类型包括公司管理层在运营策略、财务政策和投资策略方面的选择决定。其他常见个体风险形式还包括公司内部文化与实力，以及运营所在地。[2]

## 关键要点
- 个体风险特定于单一公司或股票，可通过多元化有效缓解，与系统风险无关。[2]
- 常见个体风险因素包括管理决策、公司文化、运营策略和监管事件。[2][3]
- LendingClub案例：2018年FTC指控欺诈行为导致股价下跌15%，2016年CEO离职事件均为个体风险示例。[2]
- 多元化是将投资分散到低相关性资产中，以对冲个体风险影响。[2][1]
- 系统风险（如利率、通胀）影响整个市场，无法通过多元化消除。[2][1]

## 风险与边界
- 个体风险不可预测，主要源于公司特定事件，不影响整体市场或行业。[2]
- 投资者应结合系统风险和个体风险理解，以规划多元化投资组合。[2]
