# 投资回报率（ROI）是什么及其计算方法
Language: zh-CN
Original Title: What Is Return on Investment (ROI) and How to Calculate It
Source URL: https://www.investopedia.com/terms/r/returnoninvestment.asp
Search Query: return on equity explanation

## 核心内容（中文）
投资回报率（**ROI**）是一种绩效衡量指标，用于评估投资的效率或盈利能力。它以百分比形式表示，衡量投资相对于其成本产生的利润（或损失）。

**ROI** 的计算公式为：  
\[\text{ROI} = \dfrac{\text{投资当前价值} - \text{投资成本}}{\text{投资成本}}
\]  
其中，“投资当前价值”指投资在特定时刻的价值，对于股票等类似投资，包括当前市场价值加上购买时的费用；对于项目，则包括投资生命周期的折现现金流及维护成本。“投资成本”指购买投资时的总支出，包括费用和项目融资时的成本。

**ROI** 因其通用性和简单性而广受欢迎，可作为投资盈利能力的初步衡量工具。它适用于股票投资、工厂扩建或房地产交易等场景。计算简单，便于解释：如果 **ROI** 为正值，则投资通常值得；若有更高 **ROI** 的机会，则可优先选择；负 **ROI** 表示净损失，应避免。

例如，假设Jo在2017年投资1000美元于Slice Pizza Corp.，一年后以1200美元出售。净收益为200美元，**ROI** = 200 / 1000 = 20%。相比之下，Jo在2014年投资2000美元于Big-Sale Stores Inc.，2017年以2800美元出售，**ROI** = 800 / 2000 = 40%。比较时需考虑持有年限，第一笔为1年，第二笔为3年。

**ROI** 的关键影响因素包括初始投资额、持续维护成本及投资产生的现金流。由于以百分比表示，便于跨不同投资类型比较。

**ROI** 常与其他指标如内部收益率（**IRR**）和净现值（**NPV**）结合使用，以进行更深入分析。它不考虑时间、通胀或风险，因此在比较时需谨慎。

## 关键要点
- **ROI** 衡量投资相对于成本的利润，以百分比表示，便于跨投资比较[1][2]。
- 计算公式：(投资当前价值 - 投资成本) / 投资成本[1][2]。
- 适用于股票、项目、房地产等多种投资场景，简单易用[1][5]。
- 正 **ROI** 表示盈利，负值表示损失，但需结合时间因素评估[1][4]。

## 风险与边界
- **ROI** 不考虑持有时间、通胀或风险，导致跨期比较不准确，如短期高 **ROI** 可能优于长期但年化更低者[1][4]。
- 建议与其他指标（如 **IRR**、**NPV**）结合，避免单一依赖[1]。
