# 投资回报率（ROI）：含义与计算公式
Language: zh-CN
Original Title: ROI: Return on Investment Meaning and Calculation Formulas
Source URL: https://www.investopedia.com/articles/basics/10/guide-to-calculating-roi.asp
Search Query: return on invested capital explanation

## 核心内容（中文）
**投资回报率（ROI）**是一种比率，用于衡量投资的盈利能力，通过将收益或损失与投资成本进行比较。它有助于评估股票或商业项目的潜在回报，通常以百分比形式呈现，并可使用特定公式计算。[1][3]

ROI的计算有两种方法。第一种方法：  
\[ \text{ROI} = \frac{\text{投资净回报}}{\text{投资成本}} \times 100\% \]  
第二种方法：  
\[ \text{ROI} = \frac{\text{FVI} - \text{IVI}}{\text{投资成本}} \times 100\% \]  
其中，**FVI**为投资最终价值，**IVI**为投资初始价值。[1][2][5]

ROI通常以百分比表示，便于理解，其分子包含净回报，因为投资回报可能为正或负。要实现最高准确度的ROI计算，应考虑总回报和总成本；在比较竞争性投资时，应考虑**年化ROI**。[1][6]

当ROI计算结果为正值时，表示净回报为正（总回报超过总成本，即“盈利”）；结果为负值时，表示净回报为负（总成本超过总回报，即“亏损”）。ROI公式看似简单，但依赖于成本的准确核算，对于股票简单，但对于拟议商业项目则更复杂。[1]

**ROI示例**：假设投资者以每股10美元买入1000股虚构公司Worldwide Wickets Co.股票，一年后以12.50美元卖出，持有期内获得500美元股息，并支付125美元交易佣金。  
计算：  
\[ \text{ROI} = \frac{(12.50 - 10) \times 1000 + 500 - 125}{10 \times 1000} \times 100 = 28.75\% \]  
总回报包括资本利得和股息，总成本包括初始购买价和佣金。资本利得贡献25%，股息贡献5%，佣金扣减1.25%。[1]

另一种分解公式：**ROI = 资本利得% - 佣金% + 股息收益率**。资本利得 = (2500 ÷ 10000) × 100 = 25.00%；佣金 = (125 ÷ 10000) × 100 = 1.25%；股息收益率 = (500 ÷ 10000) × 100 = 5.00%；总ROI = 25.00% - 1.25% + 5.00% = 28.75%。资本利得和股息的区分重要，因为它们税率不同。[1]

## 关键要点
- ROI是衡量投资盈利能力的比率，通常以百分比表示，通过净回报除以投资成本计算。[1][3]
- 计算公式包括净回报法和最终价值减初始价值法，需考虑总回报（如资本利得、股息）和总成本（如佣金）。[1][2][5]
- ROI适用于评估股票盈利、商业收购决策或房地产交易成功度。[1]
- 正ROI表示盈利，负ROI表示亏损；年化ROI用于比较不同持有期投资。[1][6]

## 风险与边界
- ROI不考虑投资持有时间长短，可能导致不同期限投资比较失真，建议使用年化ROI补充。[1][6]
- 计算依赖准确成本核算，对于复杂项目（如商业投资）易出错，不包括通胀或机会成本。[1]
