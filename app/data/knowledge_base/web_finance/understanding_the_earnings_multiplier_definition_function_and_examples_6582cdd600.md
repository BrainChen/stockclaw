# 收益倍数：定义、功能与示例

Original Title: Understanding the Earnings Multiplier: Definition, Function, and Examples

## 核心内容（中文）
收益倍数是一种财务估值指标，用于将公司当前股价与每股收益（**EPS**）联系起来，以判断股票是否相对于其盈利潜力被高估或低估[1]。

收益倍数的计算公式为股价除以每股收益，也称为**市盈率（P/E）**。这一指标帮助投资者比较当前股价与历史股价基于盈利的表现，并作为简化工具评估类似公司股票的相对昂贵程度[1][3]。

在股票估值中，收益倍数显示当前股价相对于公司每股收益的高低关系。股价理论上应反映公司未来价值及持有股票产生的未来现金流。如果股价相对于盈利历史性高企，可能表明此时购买股票并非最佳时机，因为其过于昂贵[1]。

比较类似公司的收益倍数，能突出各公司股价相对于彼此的昂贵程度。例如，假设虚构公司ABC当前股价为50美元，每股收益为5美元，则收益倍数为50/5=10。这意味着以当前EPS，需要10年才能收回50美元的股价[1]。

同样可以说“公司ABC以10倍盈利交易”，因为50美元是5美元EPS的10倍。10年前，若公司ABC股价同样为50美元但EPS为7美元，则倍数为7.14。现在的10倍高于当时的7.14倍，表明当前股价相对于盈利更昂贵[1]。

将公司ABC的收益倍数与其他公司比较有助于评估相对昂贵度。若公司XYZ EPS也为5美元但股价为65美元，其倍数为13，则其股票相对于公司ABC更昂贵。收益倍数仅适用于相对估值，不应用于绝对估值分析[1][3]。

## 关键要点
- 收益倍数（**P/E**）通过股价除以**EPS**计算，用于评估股票相对于盈利的定价公平性[1][2][3]。
- 高收益倍数可能表示股票昂贵或预期强劲增长，低倍数可能表示低估[1][3][6]。
- 适用于比较类似公司或历史表现，但不适合亏损公司或绝对估值[1][3]。
- 示例：股价50美元、EPS 5美元，倍数为10，意味着投资者为每1美元盈利支付10美元[1][7]。

## 风险与边界
- 收益倍数仅为相对估值工具，不应用于绝对估值决策，可能忽略未来增长或行业差异[1][3]。
- 对于亏损公司，负P/E无意义，且高倍数可能受预期增长、市场情绪影响而非当前盈利[3][6]。
