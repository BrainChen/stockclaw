# 市净率（市账率）- 定义、公式

## 核心内容（中文）
**市净率**（也称**市账率**，Market to Book Ratio 或 Price to Book Ratio）是一种财务估值指标，用于评估公司当前市场价值与其账面价值的相对关系。市场价值指所有流通股当前股价的总和，即市场认为公司价值的价格；账面价值指公司清算所有资产并偿还所有负债后剩余金额。

账面价值等于公司净资产，来源于资产负债表。换言之，该比率用于比较公司可用净资产与其股票销售价格的关系。

市净率通常由投资者使用，以显示市场对特定股票价值的看法。它常用于评估保险公司、金融公司、房地产公司和投资信托公司，但不适用于主要拥有无形资产的公司。该比率表示投资者为每美元净资产支付多少权益。

市净率的计算方法是将股票当前收盘价除以最近季度每股账面价值。公式为：**市值 / 净账面价值** 或 **股价 / 每股净账面价值**，其中净账面价值 = 总资产 - 总负债。

比率低于1可能表示股票被低估（即不良投资），高于1可能表示股票被高估（即表现良好）。但许多观点相反，由于意见分歧，建议结合或替代使用其他股票估值方法。公司低比率也可能表示公司存在问题，或支付过多以换取破产时剩余价值。

市净率帮助公司判断其资产价值是否与股票市场价格相当。最佳实践是在同一行业内比较公司间的市净率。

## 关键要点
- **计算公式**：市净率 = 股价 / 每股净账面价值，或市值 / 净账面价值（净账面价值 = 总资产 - 总负债）。
- **适用行业**：保险公司、金融公司、房地产公司和投资信托；不适合无形资产为主的公司。
- **Excel 示例**：股票1的市净率为3.9x（市值高于净账面价值）；股票2为0.9x（市值低于净账面价值）。
- **公式推导**：市净率 = **PE × ROE**，进一步为 (ROE – g) / (r – g)，其中r为权益成本，g为增长率；若ROE > r，则比率高于1。

## 风险与边界
- 比率解释存在争议，低比率可能表示低估或公司问题，高比率可能表示高估或良好表现，需结合其他估值方法。
- 不适用于无形资产占比高的公司，且最佳用于同行业比较，避免跨行业误判。
