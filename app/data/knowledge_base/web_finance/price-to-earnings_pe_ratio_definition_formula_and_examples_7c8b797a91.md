# 市盈率（P/E比率）：定义、公式与示例

Language: zh-CN
Original Title: Price-to-Earnings (P/E) Ratio: Definition, Formula, and Examples
Source URL: https://www.investopedia.com/terms/p/price-earningsratio.asp

## 核心内容

**市盈率的定义与作用**

市盈率（P/E比率）是将公司股价与每股收益（EPS）进行比较的指标[2][3]。它表示投资者为获得公司每一美元的收益需要支付多少美元[3]。例如，当P/E比率为5时，意味着投资者为公司每赚取1美元而支付5美元[3]。分析师和投资者使用这一指标来确定公司股票的相对价值，并进行并排比较[1]。

**计算公式与方法**

P/E比率的计算公式为：P/E比率 = 股价 / 每股收益（EPS）[2][3]。计算过程简单直接——只需将当前股价除以EPS即可得出结果[3]。例如，若公司股价为200美元，EPS为20美元，则P/E比率为200÷20=10，表示投资者为公司每一美元的EPS支付10美元[3]。

**P/E比率的类型**

P/E比率主要分为三种类型[2]。**静态P/E比率**基于最近一个财政年度的每股收益；**滚动P/E比率**（也称追踪P/E）基于过去12个月的净利润计算，例如上市公司发布半年报后，最新的滚动P/E比率数据为今年上半年利润加上去年下半年利润[2]；**预测P/E比率**基于预测的12个月净利润计算，例如发布季度报告后，将季度利润乘以4得到年度预测利润[2]。

**P/E比率的应用与解读**

P/E比率在比较同一行业内的多家公司时最具价值，可帮助投资者了解各公司的估值情况[5]。较高的P/E比率可能表明投资者预期公司未来增长强劲，而较低的P/E比率可能表明市场对公司前景的看法不够乐观[4]。例如，A公司股价30英镑、EPS 1.50英镑，P/E比率为20；B公司股价35英镑、EPS 2.50英镑，P/E比率为14。虽然B公司股价更高，但其P/E比率更低，意味着投资者为每英镑利润支付的价格更低[4]。

**P/E比率与其他估值指标的关系**

与P/E比率密切相关的是价格/收益增长比（PEG比率），其公式为：PEG比率 = 追踪P/E比率 / 预测增长率[6]。PEG比率通过同时分析股票当前价格和预测增长率，能比P/E比率更准确地判断股票是否被高估或低估[6]。此外，P/E比率还可与历史表现、同行业其他公司或整体市场进行基准比较[1]。

**P/E比率的局限性**

无盈利或亏损的公司没有P/E比率，因为分母中没有数值可用[1]。P/E比率主要适用于利润稳定的公司估值[2]。此外，P/E比率基于历史或预测数据，可能无法完全反映公司的实际价值，需要与其他财务指标结合使用以做出更全面的投资决策。

## 关键要点

- P/E比率 = 股价 ÷ 每股收益，表示投资者为公司每一美元收益支付的价格[3]
- 主要分为静态P/E、滚动P/E（追踪12个月）和预测P/E三种类型[2]
- 较高的P/E比率可能反映投资者对未来增长的乐观预期，较低的P/E比率则可能表示估值相对便宜[4]
- P/E比率在比较同行业公司时最具价值，可帮助投资者做出更明智的投资决策[5]

## 风险与边界

- 无盈利或亏损的公司无法计算P/E比率，因为分母为零或负数[1]
- P/E比率主要适用于利润稳定的公司，对于高增长或周期性行业的公司可能不够准确，应结合PEG比率等其他指标综合分析[2][6]
