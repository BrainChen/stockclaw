# 资产负债表：解释、组成部分及示例

## 核心内容（中文）
资产负债表是财务报表的一种，提供公司资产、负债和股东权益在特定时间点的快照。

资产负债表显示公司拥有什么、欠什么以及股东投资的金额。它详细列出公司的**资产**、**负债**和**股东权益**。投资者和分析师使用它评估公司财务健康，进行基本面分析，并计算关键比率，如流动性比率、杠杆比率和**ROE**。

资产负债表是评估企业使用的三大核心财务报表之一。它为过去经营期公司的财务状况（拥有和欠款）提供快照。资产负债表遵循会计等式：**资产 = 负债 + 股东权益**。分析师利用资产负债表计算财务比率。

资产负债表提供公司财务在某一时刻的快照，但单独无法显示公司内部财务趋势，因此需与其他报表及前期报表比较。投资者可通过资产负债表衍生比率，如**debt-to-equity ratio**和**acid-test ratio**，结合损益表和现金流量表，以及财报附注评估公司财务状况。

会计等式为：\[ \text{Assets} = \text{Liabilities} + \text{Shareholders' Equity} \]。公司购买资产需通过借款（负债）或股东投资（权益）支付。例如，公司从银行借入5年期4000美元贷款，其现金账户（资产）增加4000美元，长期债务账户（负债）也增加4000美元，保持等式平衡。若公司从投资者处获得8000美元，资产和股东权益均增加。

资产负债表中资产应始终等于负债加股东权益，若不平衡，可能存在数据错误、库存或汇率问题，或计算失误。各类别包含多个子账户，依行业而异。公司可使用**common size**资产负债表，以百分比显示数值，便于比较。

## 关键要点
- 资产负债表反映特定时间点公司资产、负债和股东权益，提供财务快照。
- 遵循**资产 = 负债 + 股东权益**等式，用于计算流动性、杠杆和**ROE**等比率。
- 资产按流动性顺序排列，分流动资产（1年内变现）和非流动资产（长期资产）。
- 需与其他报表和前期数据比较，以分析趋势和行业差异。
- 常见流动资产包括现金等价物、可流通证券、应收账款、库存和预付费用。

## 风险与边界
- 资产负债表仅为静态快照，无法单独显示趋势或现金流动态，需结合损益表和现金流量表使用。
- 不同行业会计实践差异大，同一术语含义可能不同，比较时需考虑行业特性。
