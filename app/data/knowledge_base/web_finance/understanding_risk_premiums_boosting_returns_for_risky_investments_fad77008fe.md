# 理解风险溢价：提升高风险投资回报
Language: zh-CN
Original Title: Understanding Risk Premiums: Boosting Returns for Risky Investments
Source URL: https://www.investopedia.com/terms/r/riskpremium.asp
Search Query: equity risk premium explanation

## 核心内容（中文）
风险溢价代表投资超过**无风险利率**的额外回报。它补偿投资者承担的额外风险，类似于高风险工作中危险津贴。高风险投资如股票或不稳定债券，必须承诺更高的潜在回报，以证明潜在损失的风险。

风险溢价类似于投资的危险津贴，补偿损失风险。高风险投资必须提供更大回报潜力，以补偿投资者可能损失部分或全部资本的风险。这种补偿形式是风险溢价，即高于无风险投资（如美国政府证券）回报的额外部分。该溢价奖励投资者面对失败企业损失资金的前景，且只有企业成功时才实际获得。

风险溢价可视为奖励，因为成功的**高风险投资**往往更盈利。深度渗透市场的投资结果可预测，不太可能改变世界。相反，范式转变的突破更可能来自新型高风险举措。这些投资可能提供优越回报，企业家可据此奖励投资者。这种激励促使一些投资者寻求高风险投资，以期获得更大回报。

借款人面临不确定前景时，常需支付高昂风险溢价，导致更高利率。承担更多债务会增加失败和违约风险。投资者应考虑要求多少风险溢价，否则违约时可能陷入债务追收。在破产中，投资者往往仅收回每美元几美分，尽管承诺了高风险溢价。

**股权风险溢价（ERP）**指投资股票市场超过无风险利率的超额回报。该超额回报补偿投资者承担股票较高风险。溢价大小取决于特定投资组合的风险水平，并随市场风险波动而变化。高风险投资通常获更高溢价。大多数经济学家同意，长期来看，市场补偿股票投资更高风险。

ERP常用**资本资产定价模型（CAPM）**估算：CAPM（股权成本）= Rf + β (Rm - Rf)，其中Rf为无风险回报率，β为股票市场贝塔系数，Rm - Rf为市场预期超额回报。从1928年至2022年，美国ERP平均约5.06%，高于1871-1925年的4.6%和1802-187年的2.9%。

经济学家承认市场存在股权溢价，但对其存在原因同样困惑，这被称为**股权溢价之谜**。

## 关键要点
- 风险溢价补偿投资者高于**无风险利率**的额外风险，提供高风险投资回报[1][2]。
- **ERP**反映股票投资超过无风险利率的超额回报，常使用**CAPM**模型估算[3][4]。
- 1928-2022年美国ERP平均约5%，市场条件导致其波动，影响股票吸引力[1][7]。
- 高风险投资获更高溢价，借款人违约风险增加融资成本[2][5]。

## 风险与边界
- 历史ERP数据（如美国5%平均）基于过去表现，不保证未来回报，市场波动可能导致显著变化[1][3]。
- CAPM计算依赖β系数和市场假设，实际应用中存在**股权溢价之谜**，经济学家对其成因仍有争议[1][2]。
