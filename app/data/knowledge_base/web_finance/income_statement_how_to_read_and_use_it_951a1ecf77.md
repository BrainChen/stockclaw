# 损益表：如何阅读和使用它

Original Title: Income Statement: How to Read and Use It
Source URL: https://www.investopedia.com/terms/i/incomestatement.asp

## 核心内容（中文）
损益表是一种财务报告，总结了公司在特定会计期间的财务表现，包括公司的收入、费用和净利润。[1]

损益表，也称为利润与损失（P&L）表，汇总了公司在特定会计期间的收入、费用、收益和损失。该表提供了对公司运营、管理效率以及与行业同行的表现比较的宝贵洞见。[1]

损益表是公司业绩报告的重要组成部分。虽然资产负债表提供了公司特定日期财务状况的快照，但损益表报告了特定期间（通常为季度或年度）的收入情况。[1]

损益表说明了公司实现的净收入如何转化为净利润（利润或损失）。这需要报告四个关键项目：收入、费用、收益和损失。损益表从销售细节开始，逐步计算净利润，并最终得出每股收益（EPS）。在每一行中，损益表不区分现金和非现金收据（现金销售 vs. 赊销）或现金与非现金支付/支出（现金购买 vs. 赊购）。[1]

损益表涵盖的期间在其标题中标明。例如，“截至2025年1月31日的年度”表示公司财政年度，或“截至2025年3月31日的三个月”表示日历年第一季度。上市公司必须向美国证券交易委员会（SEC）提交损益表。[1]

损益表的格式可能因当地监管要求、业务范围多样性和相关运营活动而有所不同，但无论格式如何，某些信息始终存在。主要包括经营收入、非经营收入、收益和费用等部分。[1]

经营收入是通过主要活动实现的收入。对于制造产品或从事批发、分销或零售的公司，主要活动收入指产品销售收入。对于提供服务的公司，主要活动收入指提供服务所获收入或费用。[1]

非经营收入是通过次要、非核心业务活动实现的收入，来源于购销商品和服务以外的收益，如银行资本利息、租赁业务财产、战略伙伴关系（如特许权使用费）、业务财产广告等。[1]

收益，也称为其他杂项收入，表示从其他活动（如出售长期资产）获得的净收入，包括一次性非业务活动，如出售旧运输货车、闲置土地或子公司。[1]

收入不应与收据混淆。支付通常在销售或服务交付期间入账，而收据在收到现金时入账。例如，客户于9月28日获取商品或服务，该收入计入9月；若给予30天付款期，则收据于10月28日入账。[1]

企业的持续运营和盈利成本称为费用，其中一些费用若符合美国国税局（IRS）指南，可在税表中抵扣。主要活动费用是为赚取与主要活动相关的平均经营收入而产生的全部费用。[1]

## 关键要点
- 损益表从收入开始，展示收入、费用、收益和损失，最终得出净利润，提供公司运营和效率洞见。[1]
- 与资产负债表和现金流量表不同，损益表聚焦特定期间的盈利表现，可采用单步或多步格式。[1]
- 关键组成部分包括经营收入、非经营收入、收益和各种费用，不区分现金与非现金交易。[1]
- 上市公司需向SEC提交损益表，用于评估管理效率和行业比较。[1]

## 风险与边界
- 输入英文内容可能被截断，仅覆盖损益表定义、组成部分至主要活动费用，完整分析需参考原文档。[1]
- 不同国家监管（如GAAP vs. IFRS）可能导致格式差异，分析时需注意本地会计准则。[1]
