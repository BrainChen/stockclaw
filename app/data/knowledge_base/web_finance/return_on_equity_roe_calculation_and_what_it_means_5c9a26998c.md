# 股东权益报酬率（ROE）的计算与含义

Language: zh-CN
Original Title: Return on Equity (ROE) Calculation and What It Means
Source URL: https://www.investopedia.com/terms/r/returnonequity.asp

## 核心内容

**ROE的定义与本质**

股东权益报酬率（ROE）是衡量公司财务绩效的关键指标，通过将净收入除以股东权益来计算[1]。由于股东权益等于公司资产减去债务，ROE实际上反映的是公司净资产的回报率[1]。ROE被视为衡量企业盈利能力和利润生成效率的重要指标，ROE越高，说明公司管理层在利用权益融资创造收入和增长方面的效率越高[1]。

**ROE的计算方法**

ROE的标准计算公式为：ROE = 净收入 / 股东权益[1]。其中净收入是指扣除利息、税款以及优先股股息后的收益，但不包括普通股股息[1]。最佳实践是基于一段时期内的平均股东权益来计算ROE，因为收益表和资产负债表之间存在时间错配[1]。平均股东权益通过将期初和期末的股东权益相加后除以2来计算[1]。

**ROE的行业对标与评估**

ROE的好坏评估需要与同行业公司进行比较[1]。不同行业的正常ROE水平差异很大，例如公用事业公司由于资产和债务较多而收入相对较少，其正常ROE可能为10%或更低；而科技或零售公司相对于净收入的资产负债表账户较小，正常ROE水平可能达到18%或更高[1]。一般而言，ROE低于10%在许多行业可能被视为较弱，而接近或超过广泛市场平均水平的回报率则更受欢迎[1]。

**ROE与公司盈利能力的关系**

ROE直接反映了公司利用股东资本创造利润的能力[2]。较高的ROE意味着公司能够更有效地运营，为股东带来更高的回报[2]。投资者可以通过比较不同公司的ROE来识别哪些公司在运用股东资本方面表现更佳，具有更好的盈利能力和价值创造能力[2]。

**DuPont分析框架**

ROE可以通过DuPont分析进一步拆解，将其分解为三个关键因素：净利润率、总资产周转率和权益乘数[7]。具体公式为：ROE = 净利润率 × 总资产周转率 × 权益乘数[7]。其中净利润率反映公司产品或服务的竞争力，净利润率高的公司通常在行业中竞争对手较少或构建了包括品牌、技术等在内的竞争壁垒[7]。

**ROE与其他指标的区别**

ROE与资产回报率（ROA）都是衡量公司财务表现的重要指标，但关注焦点不同[1]。ROE关注股东权益的回报，而ROA关注总资产的回报[1]。ROE受公司资本结构影响较大，因为负债水平的增加会减少股东权益的比例，可能会人为提高ROE；而ROA因为计算中包括了负债，所以更能反映公司不考虑资金结构时的盈利能力[1]。

## 关键要点

- ROE通过将净收入除以股东权益计算得出，是衡量公司利用股东资本创造利润效率的关键指标
- 较高的ROE通常表示公司盈利能力更强，但过高的ROE可能由高债务水平导致
- ROE最适合用于比较同一行业内的公司，因为不同行业的正常ROE水平差异显著
- DuPont分析可将ROE分解为净利润率、资产周转率和财务杠杆三个因素，帮助深入理解盈利驱动力

## 风险与边界

- ROE可能被高债务水平人为提高，因此需要结合公司的资本结构和负债情况进行综合评估
- 单独使用ROE进行投资决策存在局限性，应与其他财务指标（如ROA、现金流等）结合使用，并考虑行业特性和公司具体情况
