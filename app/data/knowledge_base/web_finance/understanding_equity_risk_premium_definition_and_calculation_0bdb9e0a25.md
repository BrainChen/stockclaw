# 股权风险溢价：定义与计算

Language: zh-CN
Original Title: Understanding Equity Risk Premium: Definition and Calculation
Source URL: https://www.investopedia.com/terms/e/equityriskpremium.asp

## 核心内容

**股权风险溢价的基本定义**

股权风险溢价（ERP）是指投资者在投资股票相比无风险投资（如国债）时所获得的额外回报[4]。它代表了股票投资相对于国库券或债券等低风险资产的超额收益，用以补偿投资者承担的更高风险[1]。这一概念基于风险-收益权衡原理，反映了投资者对承担股市风险的合理补偿。

**股权风险溢价的理论基础**

股权风险溢价是一个前瞻性指标，虽然在理论上有效，但本质上是估计性的，而非精确科学[1]。由于无法准确预测股票或股市未来表现，股权风险溢价通常基于历史表现进行估算，通过观察特定时期内股票市场和政府债券的表现来推断未来回报的可能性[1]。不同的时间框架和计算方法会导致估算结果差异较大[5]。

**主要计算方法**

股权风险溢价有两大类主要计算方法：历史法和面向未来的方法[3]。**历史样本法**通过历史数据计算，但由于股票收益波动较大，不同时间窗口的计算结果差异明显。例如，以2005年1月至2020年7月为样本期计算的股权风险溢价为7.74%，而2010年8月至2020年7月的样本期则为3.14%[5]。**隐含风险溢价法**则通过股票市盈率倒数减去国债收益率来计算，其中市盈率倒数代表持有股票的预期收益，国债收益率代表机会成本，两者差值即为隐含风险溢价[5]。

**CAPM模型与计算步骤**

资本资产定价模型（CAPM）是计算股权风险溢价的核心工具，公式为：Ra = Rf + βa(Rm - Rf)[1]。其中Ra为股权投资的预期回报，Rf为无风险回报率，βa为衡量股票相对市场波动性的系数Beta，Rm为市场预期回报[1]。具体计算步骤包括：确定股权投资的预期回报、确定无风险收益率、计算Beta系数、估算市场预期回报，最后使用公式"股权风险溢价 = Ra - Rf"或重新整理CAPM为"βa(Rm - Rf)"[1]。

**股利贴现模型与增长模型**

Gordon模型假设股利增长率保持不变，通过公式P = D1/(r-g)来计算股票价值，其中D1为第一期股利，r为折现率（即股票预期收益率或股权资本成本），g为股利增长率[3]。通过解出r可得到r = D1/P + g，这种方法被美林高盛等机构广泛应用，用于估算数百家公司的预期收益率[3]。另一种方法是使用收益收益率（市盈率的倒数）来估算预期回报，这对前瞻性分析特别有用。

**影响股权风险溢价的关键因素**

股权风险溢价的大小受多个因素影响，包括经济基本面、投资者风险偏好和突发事件[5]。市场风险（由信息不对称导致）和企业自身经营风险都会直接影响股权风险溢价的水平[2]。此外，指数选取、无风险利率的期限选择以及计算方法的差异也会显著影响计算结果[5]。

## 关键要点

- 股权风险溢价是投资者投资股票相比无风险资产所获得的额外回报，用以补偿更高的投资风险
- 主要计算方法包括历史样本法和隐含风险溢价法，两种方法在不同市场条件下可能产生显著差异
- CAPM模型是计算股权风险溢价的标准工具，需要准确估算无风险利率、Beta系数和市场预期回报
- 股权风险溢价受经济基本面、风险偏好、指数选取和计算方法等多个因素影响，不同时间框架的估算结果可能相差很大

## 风险与边界

- 股权风险溢价基于历史数据估算，不是精确科学，无法完全准确预测未来回报，经济学家认可其概念有效性但不保证其预测准确性
- 计算结果易受幸存者偏差影响（过度关注历史表现良好的市场如美国市场），且不同时间窗口、无风险利率期限选择和计算方法的差异会导致估算结果差异较大
