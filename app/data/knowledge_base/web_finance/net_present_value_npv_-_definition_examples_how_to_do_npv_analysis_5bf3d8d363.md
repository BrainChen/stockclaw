# 净现值（NPV）：定义、示例与分析方法

**Language:** zh-CN  
**Original Title:** Net Present Value (NPV) - Definition, Examples, How to Do NPV Analysis  
**Source URL:** https://corporatefinanceinstitute.com/resources/valuation/net-present-value-npv/

## 核心内容

**净现值的定义**

净现值（NPV）是指投资整个生命周期内所有未来现金流（包括正现金流和负现金流）折现到当前时点的价值[1][4]。NPV分析是一种内在价值评估方法，广泛应用于财务和会计领域，用于确定企业价值、投资证券、资本项目、新创企业、成本削减计划以及任何涉及现金流的事项的价值。

**NPV分析的用途**

NPV分析用于帮助确定投资、项目或任何一系列现金流的价值[1]。它是一个全面的指标，因为它考虑了与投资相关的所有收入、支出和资本成本，并将其纳入自由现金流（FCF）中。除了考虑所有收入和成本外，NPV还考虑每笔现金流的时间，这可能对投资的现值产生重大影响。例如，较早看到现金流入和较晚看到现金流出，比反之更有利。

**现金流折现的原因**

净现值分析中的现金流折现主要有两个原因：（1）调整投资机会的风险，（2）考虑货币的时间价值（TVM）[1]。首先，并非所有企业、项目或投资机会都具有相同的风险水平。例如，从美国国债获得现金流的概率远高于从年轻科技初创公司获得现金流的概率。为了调整风险，风险较高的投资采用较高的折现率，而较安全的投资采用较低的折现率。美国国债被视为无风险利率，所有其他投资都根据相对于该利率承担的额外风险来衡量。其次，由于通货膨胀、利率和机会成本，货币越早收到越有价值[1]。例如，今天收到100万元远优于五年后收到100万元。如果今天收到资金，可以进行投资并获得利息，因此五年后的价值将超过100万元。

**NPV计算示例**

以一项投资为例，假设该投资在10年期间每年返回1万元，所需的折现率为10%。通过NPV公式计算，该投资的现值为61,446元[1]。这意味着理性投资者愿意今天支付最多61,446元来获得未来10年每年1万元的现金流。按此价格支付，投资者将获得10%的内部收益率（IRR）。如果支付少于61,000元，投资者将获得超过10%的内部收益率。

**NPV计算工具**

Excel提供两个用于计算净现值的函数：NPV和XNPV[1]。常规NPV函数=NPV()假设一系列中的所有现金流以固定间隔（即年、季度、月）发生，不允许这些时间段有任何变化。XNPV函数=XNPV()允许为每笔现金流应用特定日期，使其可以以不规则间隔发生。由于现金流通常分布不均，这种增强的精确度水平是必需的。

**NPV与内部收益率（IRR）的关系**

内部收益率（IRR）是使投资净现值等于零的折现率[1]。换句话说，它是投资者在投资生命周期内预期获得（或实际获得）的复合年回报率。例如，如果一种证券提供一系列现金流，NPV为5万元，投资者恰好支付5万元购买，则投资者的NPV为0，意味着他们将获得该证券上的折现率。理想情况下，投资者会支付少于5万元，因此获得大于折现率的IRR。通常，投资者和企业管理者会同时查看NPV和IRR。

**NPV的决策标准**

根据NPV的大小进行投资决策：当NPV≥0时，投资方案可行；当NPV<0时，投资方案不可行；当多个方案的NPV均>0时，NPV最大的方案为最优方案[4]。虽然NPV为负值在理论上表示投资方案不可接受，但从实际操作层面来说，这也许会与公司的战略性决策有关，比如为了支持其他项目、开发新市场和产品、寻找更多机会获得更大利润，或回避税收[4]。

## 关键要点

- **NPV是全面的价值评估指标**，它将投资的所有未来现金流折现到当前时点，扣除初始投资成本后得出净值，用于衡量投资项目的财务可行性[1][4]

- **折现率的选择至关重要**，需要同时考虑投资风险和货币时间价值，风险越高的投资应采用越高的折现率[1]

- **NPV>0表示投资可增加价值**，投资者愿意支付的价格等于NPV加上初始投资额，此时投资具有利可图的潜力[1][4]

- **NPV与IRR相辅相成**，IRR是使NPV=0的折现率，两者常被投资者和管理者同时使用以做出更全面的投资决策[1]

## 风险与边界

- **折现率的选取对NPV结果影响重大且不易正确估计**，不同的折现率假设可能导致完全不同的投资评估结论，因此需要谨慎确定合理的折现率[2]

- **NPV方法基于对未来现金流的预测**，而实际现金流往往存在不确定性，特别是对于长期项目，预测的准确性直接影响NPV分析的可靠性[1]
