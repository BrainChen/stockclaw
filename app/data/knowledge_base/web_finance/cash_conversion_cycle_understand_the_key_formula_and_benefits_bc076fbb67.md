# 现金转换周期：理解关键公式与益处
Language: zh-CN
Original Title: Cash Conversion Cycle: Understand the Key Formula and Benefits
Source URL: https://www.investopedia.com/articles/06/cashconversioncycle.asp
Search Query: working capital management basics

## 核心内容（中文）
现金转换周期（**CCC**）追踪公司将用于库存和生产的现金支出转化为从客户收到的现金所需的时间。**CCC** 越短越好，因为这表明资金在应收账款或库存中锁定的时间越少。[1][2]

**CCC** 是评估管理层使用营运资本效果的多个指标之一。营运资本是日常运营中使用的资金。该指标衡量公司将投入运营的资金转化为现金所需的时间。[1][5]

**CCC** 使用支付供应商、创建库存、销售产品和收取客户付款的平均时间。通常，时间框架越短，对公司越有利。[2][3]

**CCC** 结合了涉及未售出库存、销售、应收账款（**AR**）和应付账款（**AP**）的多个活动比率。未售出库存是尚未销售的库存，应收账款是公司需收取的账户，应付账款是公司需支付的账户。[1][4]

计算 **CCC** 需要从公司财务报表中收集信息，包括期间平均库存、销售成本、应收账款余额、年度收入和期末应付账款。这些信息用于计算存货周转天数（**DIO**）、销售周转天数（**DSO**）和应付账款周转天数（**DPO**）。[1][2][6]

**DIO** 是销售全部库存所需的天数，数字越小越好。先计算平均库存：（期初库存 + 期末库存）/ 2。然后计算 **DIO**：平均库存 / 销售成本 × 365 天。[1][4]

**DSO** 是收取销售款项所需的天数。先计算平均应收账款：（期初 **AR** + 期末 **AR**）/ 2。然后计算 **DSO**：（应收账款 / 年度收入）× 期间天数。[1][6]

**DPO** 是支付应付账款所需的天数。该指标反映公司支付账单的能力，**DPO** 越长越好，因为公司可更长时间持有现金。计算公式：期末应付账款 / （销售成本 / 天数）。[1][4]

**CCC** 计算公式：**DIO** + **DSO** - **DPO**。[1][2][6]

例如，虚构零售商 X 公司数据（单位：百万美元）：2022 年收入 9000、销售成本 3000、库存 1000（2022）至 2000（2023）、**AR** 100 至 90、**AP** 800 至 900。平均库存 1500、平均 **AR** 95、平均 **AP** 850。然后 **DIO** = (1500 / 3000) × 365 = 182.5 天；**DSO** = (95 / 9000) × 365 = 3.9 天；**DPO** = 850 / (3000 / 365) = 103.4 天；**CCC** = 182.5 + 3.9 - 103.4 = 83 天。[1]

单独使用 **CCC** 意义不大，应用于追踪公司随时间改进情况，并与竞争对手比较。结合 **ROE** 和 **ROA** 等其他指标，可评估公司效率。**CCC** 最低的公司通常资源利用更高效，但并非绝对。[1][5]

追踪多年 **CCC** 可揭示趋势，例如 X 公司 2022 年 83 天，2023 年 130 天，表明恶化。需调查外部因素如市场条件、销售影响，或检查 **DIO**、**DSO**、**DPO** 组成部分，如供应商或客户问题、原材料短缺或运输问题。[1]

与竞争对手比较，例如 X 公司 2023 年 **CCC** 130 天，Y 公司 100.9 天，Y 公司表现更好，可能 **DIO** 更低、**DSO** 更短或 **DPO** 更高。线上零售商 Z 公司 **CCC** 为负值 -31.2 天，意味着在收到销售款项后才支付供应商，从而无需持有大量现金。[1]

## 关键要点
- **CCC** 越低，表示公司运营资金转换效率越高，现金锁定时间越短。[1][2][3]
- **CCC** = **DIO** + **DSO** - **DPO**，其中 **DPO** 越高越有利。[1][4][6]
- 负 **CCC** 表示公司利用供应商信用提升现金流，如某些零售商。[1]
- 应结合多年趋势、竞争对手比较及其他指标如 **ROE**/**ROA** 使用。[1][5]

## 风险与边界
- **CCC** 仅为单一指标，不应用于孤立评估公司绩效，需结合外部市场因素和多指标分析。[1]
- 不同行业 **CCC** 基准差异大（如零售 vs 制造），负值不总是优越，可能反映激进信用政策风险。[1][3]
