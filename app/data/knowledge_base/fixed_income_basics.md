# 固定收益基础（利率债 / 信用债）

## 1) 债券定价核心

- 债券价格与到期收益率（YTM）通常反向变动
- 久期（Duration）衡量利率敏感度
- 凸性（Convexity）描述非线性修正，久期越长凸性作用越明显

## 2) 利率风险的直觉

在其他条件不变时，利率上行会压低债券价格，且长久期债券跌幅通常更大。

## 3) 信用债分析框架

- 偿债能力：杠杆率、利息覆盖、经营现金流
- 债务结构：短债压力、到期分布、再融资能力
- 契约条款：回售、赎回、交叉违约
- 外部支持：股东背景、政策属性、行业景气

## 4) 信用利差的含义

- 信用利差扩大：风险偏好下降 / 对违约风险担忧上升
- 信用利差收窄：流动性改善 / 风险偏好修复

## 5) 固收投资常见风险

- 利率风险：政策路径超预期
- 信用风险：主体现金流恶化或再融资受阻
- 流动性风险：市场交易深度下降导致折价卖出

## 6) 一句话总结

固定收益不等于“无风险”，收益本质来自对利率、信用和流动性风险的定价补偿。
