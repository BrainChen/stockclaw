# 估值基础（PE / PB / PEG / EV）

## 1) 核心定义

- **PE（市盈率）** = 股价 / 每股收益（EPS）
- **PB（市净率）** = 股价 / 每股净资产（BVPS）
- **PEG** = PE / 未来盈利增速（通常看 1 年或 2 年）
- **EV/EBITDA**：企业价值相对经营现金盈利能力，常用于资本结构差异较大的公司

## 2) 何时使用

- PE：适合盈利相对稳定、利润口径可比的行业
- PB：适合银行、保险、周期资产型行业（需要关注资产质量）
- PEG：适合成长股横向比较，但高度依赖盈利预测准确性
- EV/EBITDA：适合跨资本结构比较，受折旧政策影响较小

## 3) 常见误区

- 高 PE 不一定贵：若盈利高增速可持续，高估值可能合理
- 低 PB 不一定便宜：可能反映资产减值风险或盈利能力长期弱
- PEG=1 不是铁律：利率环境、商业模式稳定性会改变合理阈值
- 只看静态估值容易误判：需要结合盈利修正方向（上修/下修）

## 4) 实务判断框架

1. 先确认利润质量：是否存在一次性收益、会计调整
2. 再看估值位置：与历史分位、同业中位数比较
3. 最后看宏观贴现率：无风险利率上行会压低合理估值区间

## 5) 一句话总结

估值是“价格与预期”的对照，不是单一公式结论；必须结合增长质量、利率与风险溢价共同判断。
