# 财务报表基础（利润表/现金流/资产负债表）

## 1) 三张表的关系

- **利润表**：反映“赚了多少”（收入、成本、费用、净利润）
- **现金流量表**：反映“钱是否真的回来了”（经营、投资、融资现金流）
- **资产负债表**：反映“家底与杠杆”（资产、负债、所有者权益）

利润不等于现金，净利润高但经营现金流弱，可能意味着回款压力或应收上升。

## 2) 常用同比/环比口径

- 同比（YoY）：与去年同季度比较，适合观察中长期趋势
- 环比（QoQ）：与上季度比较，适合观察短期拐点
- TTM：过去四个季度合计，平滑季节性波动

## 3) 关键财务质量指标

- 毛利率 = （收入 - 成本）/ 收入
- 经营利润率 = 经营利润 / 收入
- 经营现金流净额 / 净利润（现金含量）
- 应收账款周转天数、存货周转天数

## 4) “增收不增利”常见原因

1. 产品结构变化导致毛利率下滑
2. 销售、管理、研发费用率上升
3. 汇率、减值、一次性损失影响净利
4. 行业竞争加剧导致价格压力

## 5) 财报阅读顺序建议

先看管理层指引变化，再看核心经营指标，最后验证现金流与资产质量是否匹配。
