# 宏观变量到资产价格的传导

## 1) 关键变量

- 经济增长（需求强弱）
- 通胀（成本与购买力）
- 利率（贴现率与融资成本）
- 汇率（进出口竞争力）
- 流动性（风险偏好与估值弹性）

## 2) 三条核心传导链

1. **盈利链**：宏观变化 → 企业收入/成本 → 利润变化  
2. **估值链**：利率与风险溢价变化 → 折现率变化 → 估值中枢变化  
3. **情绪链**：政策预期变化 → 风险偏好变化 → 资金风格切换

## 3) 常见场景

- 利率上行：成长股估值往往承压，价值/防御风格相对占优
- 信用扩张：周期行业盈利弹性更高，风险资产表现改善
- 通胀回落：成本压力缓解，利润率修复空间提升

## 4) 分析注意点

- 看“方向 + 幅度 + 持续性”，不要只看单月数据
- 市场通常先交易预期，再交易落地数据
- 同一宏观变量对不同行业影响可能相反

## 5) 一句话总结

宏观变量不是直接决定涨跌，而是通过“盈利预期 + 折现率 + 风险偏好”共同作用于资产定价。
